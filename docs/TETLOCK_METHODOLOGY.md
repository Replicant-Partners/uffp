# Tetlock Superforecasting Methodology

## Principles

1. **Outside View** - Start with base rates
2. **Decomposition** - Break into independent drivers
3. **Probabilistic** - Use ranges, not points
4. **Pre-Mortem** - Identify failure modes
5. **Brier Scoring** - Measure accuracy
